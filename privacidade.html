Política de Privacidade

Última atualização: Julho de 2026

A LUDEX LEMINE - Comércio e Indústria, Lda respeita a privacidade dos seus clientes e visitantes. Esta política explica de forma clara quais dados recolhemos, como os usamos e quais os seus direitos.

1. Quem Somos

A LUDEX LEMINE - Comércio e Indústria, Lda é uma empresa distribuidora grossista de bens de consumo essenciais, sediada em Angola. Este sítio web (website) serve como catálogo de produtos para clientes grossistas e parceiros comerciais.

2. Que Dados Recolhemos

Recolhemos apenas os dados que o utilizador nos fornece voluntariamente através do formulário de pedido de cotação:

  • Nome completo — para identificar o responsável pelo pedido.
  • Nome da empresa / negócio — para fins comerciais.
  • Número de telefone — para contacto de retorno e confirmação de encomenda.
  • Notas e detalhes do pedido — informações adicionais fornecidas pelo utilizador.

Este site não utiliza cookies de rastreamento nem ferramentas de análise de comportamento de terceiros (como Google Analytics).

3. Como Usamos os Seus Dados

Os dados fornecidos são utilizados exclusivamente para:

  • Processar e responder a pedidos de cotação de produtos.
  • Contactar o cliente para confirmação de detalhes da encomenda.
  • Enviar propostas comerciais relacionadas com os produtos solicitados.

Os seus dados nunca são vendidos, alugados ou partilhados com terceiros para fins de marketing.

4. Como os Dados São Transmitidos

Ao submeter um pedido de cotação neste sítio, as informações são enviadas diretamente para a nossa equipa comercial via WhatsApp, utilizando a plataforma de mensagens segura da Meta (WhatsApp Business). Nenhum dado é armazenado nos servidores deste website — o site funciona integralmente do lado do cliente (client-side).

5. Armazenamento de Dados

O site utiliza o armazenamento local do navegador (localStorage) para guardar temporariamente a lista de produtos que o utilizador adicionou ao seu pedido. Estes dados:

  • São guardados apenas no dispositivo do utilizador.
  • São eliminados quando o utilizador limpa os dados do navegador.
  • Nunca são enviados para servidores externos sem a ação explícita do utilizador.

6. Os Seus Direitos

Em conformidade com a legislação aplicável, tem o direito de:

  • Aceder aos dados que nos forneceu.
  • Retificar informações incorretas.
  • Solicitar a eliminação dos seus dados dos nossos registos.
  • Opor-se ao tratamento dos seus dados para fins comerciais.

Para exercer qualquer um destes direitos, contacte-nos diretamente através dos meios indicados abaixo.

7. Segurança

Adotamos medidas razoáveis para proteger as suas informações. No entanto, nenhum método de transmissão eletrónica é 100% seguro. Recomendamos que não partilhe informações sensíveis desnecessariamente.

Podemos atualizar esta política periodicamente. Recomendamos que a consulte regularmente. Alterações significativas serão comunicadas de forma visível nesta página.
